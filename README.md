##  Project Objective

To design a community-based recipe sharing platform that helps users:
- Discover featured recipes
- View detailed ingredients and cooking steps
- Interact with recipes through a clean and modern interface
- Learn front-end concepts like animations, layout design, and user interaction

##  How to Run the Project

1. Download or clone the repository
2. Make sure all images are placed inside the `images` folder
3. Open `index.html` in any modern web browser
4. Click on a recipe card to view its details
